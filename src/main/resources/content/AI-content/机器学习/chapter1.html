如何教会计算机识别猫？🐱

传统编程方法

if (有胡须 && 有尖耳朵 && 有四条腿) {

return "是猫";

}

问题：规则永远写不完！

折耳猫怎么办？三条腿的猫呢？侧面的猫？

机器学习方法

1. 给计算机看10000张猫的照片

2. 告诉它"这些都是猫"

3. 计算机自己总结规律

优势：自动发现各种特征！

甚至包括人类说不清的特征

机器学习的主要类型

📚

监督学习

有老师指导的学习

给机器看例子和答案
让它学会对应关系

例如：图片分类、房价预测

🔍

无监督学习

自己探索发现

只给数据，没有答案
让机器找出隐藏规律

例如：用户分群、异常检测

🎮

强化学习

在实践中学习

通过奖励和惩罚
学会最优策略

例如：游戏AI、机器人控制
