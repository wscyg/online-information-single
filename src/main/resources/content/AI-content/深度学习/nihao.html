LSTM的核心优势

LSTM通过三个门控机制巧妙地解决了传统RNN的梯度消失问题：

  • ✅ 遗忘门：选择性地遗忘不重要的历史信息
  • ✅ 输入门：选择性地记忆重要的新信息
  • ✅ 输出门：选择性地输出当前需要的信息

💡 这种设计让LSTM能够捕获长期依赖关系，适用于序列建模任务！

🔑 设计的巧妙之处

  • 线性路径：通过简单乘法更新，避免梯度消失
  • 选择性控制：每个维度独立，精确控制特征保留
  • 可学习性：反向传播自动学习遗忘策略
  • 数值稳定：Sigmoid确保输出在[0,1]范围

  • 重要情节（f ≈ 0.9）：主角名字、关键转折 → 深刻记忆

  • 无关信息（f ≈ 0.1）：路人对话、重复内容 → 自动过滤

🎚️

it - 输入门控制

决定哪些新信息值得存储

  • i ≈ 1: 存储该特征
  • i ≈ 0: 忽略该特征

作用：信息过滤器

🆕

C̃t - 候选值

通过tanh生成候选记忆

特点：

  • 值域：[-1, 1]
  • 包含当前输入的信息
  • 与输入门配合使用
🔧

tanh函数

双曲正切函数

tanh(x) = (ex-e-x)/(ex+e-x)

  • 输出范围：[-1, 1]
  • 零中心化
  • 梯度更稳定

输入门的更新机制

输入门通过两步更新细胞状态：

Ct = ft ⊙ Ct-1 + it ⊙ C̃t

这个公式结合了：

  • 📌 遗忘门的输出（选择性保留旧信息）
  • 📌 输入门的输出（选择性添加新信息）

🔄 LSTM整合过程

完整的前向传播流程

  1. 遗忘阶段：ft = σ(Wf·[ht-1, xt] + bf)
  2. 输入阶段：
    • it = σ(Wi·[ht-1, xt] + bi)
    • C̃t = tanh(WC·[ht-1, xt] + bC)
  3. 更新细胞状态：Ct = ft ⊙ Ct-1 + it ⊙ C̃t
  4. 输出阶段：
    • ot = σ(Wo·[ht-1, xt] + bo)
    • ht = ot ⊙ tanh(Ct)

# 完整的LSTM前向传播实现
import torch
import torch.nn as nn

class LSTMCell(nn.Module):
    def __init__(self, input_size, hidden_size):
        super().__init__()
        self.hidden_size = hidden_size
        
        # 定义所有门的权重矩阵
        self.W_f = nn.Linear(hidden_size + input_size, hidden_size) # 遗忘门
        self.W_i = nn.Linear(hidden_size + input_size, hidden_size) # 输入门
        self.W_C = nn.Linear(hidden_size + input_size, hidden_size) # 候选值
        self.W_o = nn.Linear(hidden_size + input_size, hidden_size) # 输出门

    def forward(self, x_t, h_prev, C_prev):
        # 拼接输入
        combined = torch.cat([h_prev, x_t], dim=1)
        
        # 1. 遗忘门
        f_t = torch.sigmoid(self.W_f(combined))
        
        # 2. 输入门
        i_t = torch.sigmoid(self.W_i(combined))
        C_tilde = torch.tanh(self.W_C(combined))
        
        # 3. 更新细胞状态
        C_t = f_t * C_prev + i_t * C_tilde
        
        # 4. 输出门
        o_t = torch.sigmoid(self.W_o(combined))
        h_t = o_t * torch.tanh(C_t)
        
        return h_t, C_t
